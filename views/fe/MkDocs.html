pip是一款非常方便的python包管理工具，本文主要介绍在windows 10下安装pip方法。 1. 下载pip安装包，地址： https://pypi.python.org/pypi/pip#downloads 注意选择tar.gz包，目前最新版本为： pip-7.1.0.tar.gz (md5, pgp) 2. 解压安装包，使用CMD进入解压目录，然后使用如下命令进行安装 python setup.py install 3. 添加windows系统环境变量，与安装python时添加的方法一样 如我的python目录是 F:\Python27\; 则添加如下2个目录到系统环境变量里： F:\Python27\;F:\Python27\Scripts; 4. 进入CMD运行pip，如下图所示即表示已安装成功
