应用状态

ng-model可以为应用数据提供状态值(invalid,dirty,touched,error)

Email:

状态：

验证通过:{{statusForm.emailName.$valid}} 验证未通过:{{statusForm.emailName.$invalid}} 焦点是否被触发:{{statusForm.emailName.$touched}} 焦点是否未被触发:{{statusForm.emailName.$untouched}} 已进行交互操作:{{statusForm.emailName.$dirty}} 还没有进行交互操作:{{statusForm.emailName.$pristine}}

$untouched True if control has not lost focus yet. $touched True if control has lost focus. $pristine True if user has not interacted with the control yet. $dirty True if user has already interacted with the control.
