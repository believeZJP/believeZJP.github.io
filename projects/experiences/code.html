	//js继承
	1.instanceof 表示的就是一种继承关系，或者原型链的结构
	2.访问一个对象的属性时，先在基本属性中找
	3.如果没有，再沿着__proto__这条链向上找，这就是原型链
	4.什么是继承
			--在原有对象的基础上，略作修改，得到一个新对象
			--不影响原有对象的功能
	如何添加继承
			---属性：call
			---方法：for in
	例子，继承的拖拽
	
	
	var user = {
		"form":"hunan",
		"name":"xiaoming"
	};
	function NewUser(){
		this.country = "China";
	}
	NewUser.call(user);
	//此时的user,三个属性都有
	user.__proto__ = NewUser;
	user instanceof NewUser //false
	
	user.__proto__ = NewUser.prototype;
	user instanceof NewUser //true
